To run this application, use the following steps:

1. Download the code from this link:
	For jclum170 -> http://team1-register.googlecode.com/files/Register.zip
2. Unzip the project.
3. Install Eclipse IDE for Java EE Developers
4. Import the project into Eclipse 
   a. File -> Import
   b. Existing Project
   c. Browse to the unzipped project.
5. Ensure that Google Plugin for Eclipse is installed
   a. Help -> Eclipse Marketplace 
   b. Search for Google Plugin for your version of Eclipse
6. Ensure that you are using JDK 1.6
   a. Eclipse -> Preferences -> Java -> Installed JREs
7. Right click on the project and chose Run As -> Web Application
8. The application starts by default here: http://localhost:8888/
   a. If the application does not start on port 8888 check the console in Eclipse
9. The list of controllers will appear, click on the appropriate one to view the application.
10.To add an entry, click "Add Address Entry"
11.To update an entry click on "Update" or click on its row
12.To delete an entry click on "Delete"
